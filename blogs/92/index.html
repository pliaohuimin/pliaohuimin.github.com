Vim缩进设置

参考网站：http://linux-wiki.cn/wiki/zh-hans/Vim%E4%BB%A3%E7%A0%81%E7%BC%A9%E8%BF%9B%E8%AE%BE%E7%BD%AE

常用设置

缩进设置最常用的是：

:filetype indent on

它将开启针对文件具体类型的缩进。

下面的选项需要在命令模式下用set设置的选项的含义。使用方法如：

set ci # 开启cindent

set noet # 关闭expandtab

set sw=4 # shiftwidth=4

上述设置可以写到~/.vimrc中：

filetype indent on

set ci

set noet

set sw=4
